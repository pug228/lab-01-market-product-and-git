## Roles and responsibilities

QA - Tests the product, finds bugs, and ensures quality.

Telegram developer - Builds and maintains Telegram bots and mini apps.

Back-end engineer - Develops server logic, APIs, and databases.

Front-end developer - Implements user interfaces and client-side logic.

UI/UX designer - Designs user flows, layouts, and visual style.

## Common skills across roles

QA - Test design, manual testing, understanding of APIs.

Telegram developer - API, Mini Apps, JavaScript or Python, APIs.

Back-end engineer - Server-side programming, REST APIs, databases.

Front-end developer - HTML, CSS, JavaScript, frameworks, APIs.

UI/UX designer - Users,, Figma, others.
